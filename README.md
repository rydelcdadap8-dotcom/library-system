# library-system

## Description

This is a responsive web application designed to help users register, search, and borrow books. Users can view their borrowing history and manage their accounts. The system also includes an admin panel for librarians to add, edit, or remove books.

## Features

*   User Registration and Authentication
*   Book Search and Catalog Browsing
*   Book Borrowing and Returning
*   Borrowing History
*   User Account Management
*   Admin Panel for Librarians (Add, Edit, Remove Books)
